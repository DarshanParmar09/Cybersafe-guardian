# 🛡️ CyberSafe Guardian

## 📖 About the Project

CyberSafe Guardian is a modern cybersecurity awareness and security toolkit designed to help users identify common cyber threats and improve their online safety. The platform combines educational content with practical security tools in an intuitive, responsive web interface.

## ✨ Features

* Password Strength Analyzer
* Phishing URL Detection
* File Security Checker
* AI-Powered Cybersecurity Chatbot
* Modern Responsive UI
* Interactive Animations and Cyber-Themed Design
* Contact Form with Data Storage
* Security Awareness Content and Best Practices

## 🛠️ Services

* Analyze password strength and provide security recommendations.
* Detect potentially malicious or phishing URLs.
* Scan uploaded files for basic security analysis.
* Answer cybersecurity-related questions using an AI chatbot.
* Allow users to contact the administrator through an integrated contact form.
* Promote cybersecurity awareness through educational resources.

## 💻 Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript
* Responsive Design
* Glassmorphism UI
* CSS Animations

### Backend

* Google Apps Script (Contact Form Processing)
* Google Sheets (Stores Contact Form Submissions)
* AI API Integration (Chatbot)

## 🔒 Security Note

The AI chatbot requires API credentials to function. These credentials have been removed from the public source code for security reasons. Therefore, all project features work correctly except the chatbot, which will display an error unless valid API keys are configured.
